ML optimization, locally.

Hyperparameter search, model fitting, and explainability — on your machine, from a single config file.

Get started Open dashboard
Terminal
# install
pip install corter-ml
# configure & run
corter init
corter run data.csv --web

Features

Hyperparameter search

Random, SciPy global, and local strategies with parallel trials and early stopping.

Explainability

Permutation importance and optional SHAP when the xai extra is installed.

Live dashboard

Watch trials, scores, and insights update in the browser during a run.

How it works

1

Create config

Run corter init to generate config.yaml.

2

Optimize

Point at your CSV: corter run data.csv.

3

Review

Results land in results.json; use the dashboard for live runs.
